Notification Plugin Readme

Overview

The Notification Plugin provides the ability to push notification to Android Application.

Installation

Copy notification.jar into the plugins directory of your Openfire server. The plugin will then be automatically deployed. To upgrade to a new version, copy the new notification.jar file over the existing file.

Configuration

Now, notification does not need any configurations.

Using the Plugin

To administer users, submit HTTP requests to the notification. The service address is [hostname]plugins/notification/push-notification.jsp. For example, if your server name is "example.com", the URL is http://example.com/plugins/notification/push-notification.jsp
